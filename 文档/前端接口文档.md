# 前端接口文档

本项目前端所有 API 请求均封装在 `frontend/src/api/` 目录下，使用 Axios 实例 (`http.js`) 发起请求。
基础 URL: 由 `http.js` 根据环境自动判断，开发环境默认为 `http://localhost:10002/api`。

## 1. 认证 (auth.js)
- `login(data)`: POST `/auth/login`
- `register(data)`: POST `/auth/register`
- `logout()`: POST `/auth/logout` (同时清除本地 Token)

## 2. 商品 (products.js)
- `getProducts(query)`: GET `/products`
- `getProduct(id)`: GET `/products/{id}`
- `getProductImages(id)`: GET `/product-images/{id}`
- `createProduct(data)`: POST `/products` (Admin)
- `updateProduct(id, data)`: PUT `/products/{id}` (Admin)
- `deleteProduct(id)`: DELETE `/products/{id}` (Admin)

## 3. 分类 (categories.js)
- `getCategories()`: GET `/categories`
- `getAdminCategories(params)`: GET `/categories/admin` (Admin)

## 4. 购物车 (cart.js)
- `getCart()`: GET `/cart`
- `addToCart(item)`: POST `/cart/items`
- `updateCartItem(productId, qty)`: PUT `/cart/items/{productId}`
- `removeCartItem(productId)`: DELETE `/cart/items/{productId}`
- `clearCart()`: DELETE `/cart`

## 5. 订单 (orders.js)
- `submitOrder(data)`: POST `/orders`
- `getOrders()`: GET `/orders`
- `getOrder(id)`: GET `/orders/{id}`
- `getAdminOrders(params)`: GET `/orders/admin` (Admin)
- `updateOrderStatus(id, status)`: PATCH `/orders/{id}/status` (Admin)

## 6. 用户 (users.js)
- `getCurrentUser()`: GET `/users/me`
- `getUsers(params)`: GET `/users/admin` (Admin)

## 7. 地址 (addresses.js)
- `getAddresses()`: GET `/addresses`
- `createAddress(data)`: POST `/addresses`
- `updateAddress(id, data)`: PUT `/addresses/{id}`
- `deleteAddress(id)`: DELETE `/addresses/{id}`

## 8. 收藏 (favorites.js)
- `getFavorites()`: GET `/favorites`
- `addFavorite(productId)`: POST `/favorites`
- `removeFavorite(productId)`: DELETE `/favorites/{productId}`
- `checkFavorite(productId)`: GET `/favorites/check`

## 9. 统计 (stats.js)
- `getDashboardStats()`: GET `/stats`
