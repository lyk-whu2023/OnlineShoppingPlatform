前端（Vue3）项目结构示例
frontend/
├── public/                          # 静态资源，不经过webpack处理
│   ├── index.html
│   └── favicon.ico
├── src/
│   ├── assets/                     # 静态资源，经过webpack处理
│   │   ├── images/
│   │   ├── styles/
│   │   └── fonts/
│   ├── components/                 # 公共组件
│   │   ├── common/                 # 全局通用组件（如按钮、输入框等）

│   │   └── ui/                     # 基础UI组件
│   ├── views/                      # 页面级组件（路由组件）
│   │   ├── Home.vue
│   │   ├── Login.vue

│   │   └── ...

│   │   │   ├── user.js
│   │   │   └── ...
│   │   └── index.js                # 入口文件
│   ├── router/                     # 路由配置
│   │   ├── index.js
│   │   └── routes.js               # 路由定义

│   │   │   ├── userApi.js
│   │   │   ├── authApi.js
│   │   │   └── ...

│   │   └── ...
│   ├── utils/                      # 工具函数

│   ├── vite.config.js
│   └── ...

├── package-lock.json

└── README.md

前端请求
    ↓
Controller层 (接收请求，参数校验)
    ↓ (调用Service，传入DTO)
Service层 (业务逻辑处理)
    ↓ (调用Mapper，传入Entity)
Mapper层 (数据库操作)
    ↓ (操作Entity)
数据库
    ↓
Mapper层 (返回Entity)
    ↓ (返回Entity给Service)
Service层 (处理业务，Entity→DTO转换)
    ↓ (返回DTO给Controller)
Controller层 (包装响应)
    ↓
返回给前端
