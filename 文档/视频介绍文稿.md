# 视频介绍文稿：在线购物平台 (OnlineShoppingPlatform)

## 1. 开场 (0:00 - 0:30)

**[画面]**: PPT 封面，展示项目名称“OnlineShoppingPlatform”、课程名称“开源软件生态与开发技术”、开发者姓名及学号。

**[旁白]**:
老师好，大家好。我是2023级软件工程专业的[你的姓名]。
今天非常荣幸向大家展示我的期末实验作品——**OnlineShoppingPlatform**。这是一个基于 B/S 架构，采用当下主流的 **Spring Boot** 和 **Vue 3** 技术栈开发的企业级在线购物平台。

本项目旨在模拟真实的电商业务场景，实现了从用户浏览、选购、下单到后台管理的全流程功能。

## 2. 技术架构与设计 (0:30 - 1:00)

**[画面]**: 展示系统架构图（前后端分离架构图，包含 Spring Boot, MySQL, Vue 3, Element Plus 等图标）。

**[旁白]**:
在技术选型上，我严格遵循了企业级开发的标准：
*   **后端**：核心采用 **Spring Boot 3** 框架，利用 **MyBatis** 进行高效的数据持久化，数据库选用 **MySQL 8.0**。同时，使用 **JWT** 技术实现了安全无状态的用户认证。
*   **前端**：基于 **Vue 3** (Composition API) 框架，配合 **Vite** 极速构建。UI 方面使用了 **Element Plus** 组件库，并集成 **ECharts** 进行数据可视化展示。

## 3. 核心功能演示：用户端 (1:00 - 3:00)

**[画面]**: 切换到浏览器，展示系统首页。进行操作演示。

**[旁白]**:
接下来，让我们看看系统的实际运行效果。
首先是**用户端**。
*   **(首页)**: 首页展示了轮播图、商品分类导航以及热销商品推荐。界面设计简洁明快，响应速度极快。
*   **(商品详情)**: 点击任意商品进入详情页，这里可以看到高清大图、价格、库存以及详细描述。
*   **(购物车)**: 我们将几件商品加入购物车。进入购物车页面，可以方便地调整数量或移除商品。
*   **(下单)**: 点击“去结算”，系统会引导我们确认收货地址。确认无误后提交订单，后台会实时扣减库存。
*   **(个人中心)**: 在“我的订单”中，可以查看到刚刚生成的订单，状态显示为“待支付”。这里也支持对个人信息和地址的管理。

## 4. 核心功能演示：管理端 (3:00 - 4:30)

**[画面]**: 退出登录，使用管理员账号 (admin) 登录。跳转到后台管理界面。

**[旁白]**:
现在，让我们切换到**管理员视角**。
*   **(仪表盘)**: 登录后台，首先看到的是数据仪表盘。这里使用 ECharts 直观地展示了商品总数、用户数、订单数以及销售总额，还有热销商品排名的柱状图。
*   **(商品管理)**: 在“商品管理”模块，管理员可以上架新商品，上传图片，或者修改现有商品的价格和库存。
*   **(订单管理)**: 管理员可以查看平台所有订单。对于用户的待发货订单，管理员点击“发货”后，用户端的订单状态会同步变为“已发货”。
*   **(其他)**: 此外，系统还提供了分类管理和用户管理功能，实现了对平台资源的全面管控。
